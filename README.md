Project Overview
This project automates the monthly customer credit reconciliation process using Advanced Excel techniques.
It integrates multiple transaction files and applies lookup-based automation to ensure accurate credit posting for each customer based on ID and tier mapping.
The dashboard provides a transparent summary of total credits, adjustments, and discrepancies, enabling faster and error-free reconciliation.

Tools & Functions Used

Excel Functions: VLOOKUP, INDEX-MATCH, IFERROR, SUMIFS, Conditional Formatting

Automation: Power Query for data consolidation and cleanup

Visualization: Pivot Tables and interactive dashboards

Validation: Formula-driven error tracking and exception highlighting

Key Features

Linked multiple credit files using lookup and mapping logic for seamless reconciliation

Reduced manual credit posting time by over 60% through automated formula workflows

Ensured 100% data accuracy with built-in validation templates and conditional alerts

Created a dynamic Excel dashboard summarizing total credits, tier-level distribution, and discrepancies

Business Impact

Improved reconciliation speed and accuracy for finance and CRM teams

Eliminated manual data matching errors and redundant reporting tasks

Enhanced transparency and audit readiness through structured reporting
